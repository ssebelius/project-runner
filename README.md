Project Runner
==============

Endless runner mash up inspired by a number of games

* Sonic 2 Special Stages
 	* http://www.youtube.com/watch?v=9-wZyOahzvQ
 	- Behind character halfpipe
* Pitfall iOS
	- http://www.youtube.com/watch?v=AmZ5yaxhVac
	- Behind character w/ camera switch to angled side camera
* Running w/Friends, Running w/Bulls
	* Traditional rear-view endless runner
